# Music shop app

Music shop app based on Django, DRF, React, Redux<br/><br />

## Installation
1. Clone repository: <code>git clone https://github.com/Neon20179/musicShop.git</code>.<br/>
2. <code>cd musicShop</code>.<br/>
3. Create venv: <code>py -m venv env</code>.<br/>
4. Activate it: <code>.\env\Scripts\activate</code>.<br/>
5. Install python dependencies: <code>pip install -r requirements.txt</code>.<br/>
6. Install js dependencies: <code>yarn install</code> or <code>npm install</code>.<br/>
7. <code>cd shopApp</code>.<br/>
8. Make migrations: <code>py manage.py makemigrations</code>.<br/>
9. Migrate: <code>py manage.py migrate</code>.<br/>
10. Create admin: <code>py manage.py createsuperuser</code>.<br/>
11. Run server: <code>py manage.py runserver</code>.<br/>
12. Go to <code>http://127.0.0.1:8000/admin/</code>, log in and add something. <br/>
13. Go to <code>http://127.0.0.1:8000/</code>.<br/>
14. Enjoy!<br/>
